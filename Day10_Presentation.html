PART 2 — PRACTICAL

The Practical Task

One task per participant, observed by the instructor. You will be assigned one of these three tasks. All are things you have already done this week.

🔌
Task A — Wire a Plug Top
Wire a 3-pin plug top correctly from a stripped cable. Brown to Live, Blue to Neutral, Green/Yellow to Earth. Tighten all terminals. Replace cover and fuse. Continuity test all three pins.
Assessed: polarity, termination, fuse, continuity
🔍
Task B — Diagnose a Fault
Given a pre-faulted breadboard or PCB circuit, use the multimeter to identify the fault, name the root cause, and state the correct fix. You have 10 minutes and the multimeter.
Assessed: method, diagnosis accuracy, fix proposed
🧲
Task C — Measure & Identify
Given a set of three mystery components, identify each using the multimeter and colour code reference. Measure resistance of one, test continuity of another, identify LED polarity on a third.
Assessed: tool use, identification accuracy, safety
📋 The instructor observes and scores in real time on a 20-point rubric: safety handling (5), correct technique (10), correct result (5). You do not need to be fast — you need to be correct and safe.
